## Quick start guide

### Prerequisites

- **Node** `10.5+`
- **Yarn** `1.9+`

### Initialize

_run once to setup project_

`./init_project.js`

### Startup

_convenience script to start server & app in parallel_

`./start.sh`

## Info

- **App** bootstrapped with `create-react-app`
- **Server** bootstrapped with `graphql-boilerplate`
